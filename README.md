# 📊 E-commerce Sales Dashboard (Excel)

## 🔍 Problem
Analyze sales by product, country, and gender.

## 📁 Data
- Product
- Price
- Quantity
- Country
- Order Date
- Gender

## 📊 Dashboard
- Sales KPI
- Charts
- Map
- Slicers (Gender, Order Mode)
